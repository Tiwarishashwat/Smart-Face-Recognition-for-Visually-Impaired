# Smart-Face-Recognition-for-Visually-Impaired
This is a project based on opencv library , consisting of computer vision, machine learning and deep learning algorithms.
Steps to use this file:
Install python version 3.7.6.
Download this repository to your personal PC.
Open the project folder.
Now open cmd(if you are in windows, else open the terminal in same directory irrespective of the platform).
Now type "pip install opencv"
Hit Enter and wait for it to install.
Run the command "pip install numpy".
Hit Enter and wait for it to install.
Now you are ready to run the code.
Run the command "python SmartFaceRecognition.py".
Follow up the instructions and you would be good to go!
